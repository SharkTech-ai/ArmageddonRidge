# Security Policy

Armageddon Ridge is currently a static Blazor WebAssembly game with no server API, accounts, payments, or user-generated network content.

## Reporting

Please report security issues privately to the repository owner rather than opening a public issue. Include:

- A short description of the issue.
- Steps to reproduce.
- Affected browser/runtime versions when relevant.
- Any suggested fix or mitigation.

## Scope

In scope:

- Build or deployment files that could expose secrets.
- Browser storage misuse that could leak local data.
- Supply-chain or dependency configuration risks.

Out of scope:

- Gameplay balance bugs.
- Cosmetic rendering issues.
- Local development warnings that do not affect deployed static assets.
